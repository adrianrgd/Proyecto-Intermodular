# Proyecto-Intermodular-DAW - Adrián Rangel

## Descripcion del Proyecto:

### ¿Qué es?

- Se trata de una web/app que ayudara a gestionar en un mismo lugar todos los archivos, mods, texturas y recursos creados
por los usuarios de la comunidad de todo internet para juegos por todo el mundo.

### ¿Para qué sirve?

- Sirve para tener un lugar centralizado donde poder encontrar y descargar recursos para juegos, sin tener que buscar en
muchas páginas diferentes, toparte con paginas dudosas y colaborar en la preservacion de todo este grandioso contenido.

### ¿Que problemas soluciona?

- Las busquedas infinitas, la frustracion de no encontrar lo que buscas, la perdida de contenido por el paso del tiempo y la falta de un lugar centralizado para encontrar y descargar recursos para juegos de forma segura y eficiente.
