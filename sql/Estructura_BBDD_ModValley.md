# Estructura de base de datos ModValley


## Interaccion entre ellas

**Usuario:**
> Puede crear uno o muchos recursos (de un tipo de categoria y un tipo de juego).
> Puede escribir uno o muchos comentarios.
> Puede valorar uno o muchos recursos.
> Puede descargar uno o muchos recursos.

**Recurso:**
> Puede tener uno o muchos comentarios.
> Puede obtener una o muchas valoraciones.
> Puede tener una o muchas descargas.

**Comentarios:**
> Pueden ser escritos por uno o muchos usuarios.
> Pueden tenerlo uno o muchos recursos.

**Valoraciones:**
> Pueden ser escritas por uno o muchos usuarios.
> Pueden tenerlo uno o muchos recursos.

**VideoJuego:**
> Puede tener uno o muchos recursos.

**Categoria:**
> Puede tener uno o muchos recursos.

**Descargas:**
> Pueden realizarlas uno o más usuarios.
